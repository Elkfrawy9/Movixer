# Movixer
a sample Android project using The Movie DB API based on MVVM architecture

# Tech stack & Open-source libraries
- Kotlin
- Single activity
- Coroutines
- Hilt for dependency injection.
- Retrofit & OkHttp3
- Glide
- Paging3
- ViewPager
- Clean Architecture & Repository Pattern
